# CSV-Column-To-Row

MySQL で CSV(カンマ区切り)のカラムを行に変換する

## 例

CSV カラムのテーブル

```
mysql> select * from csv;
+----+-------+
| id | col   |
+----+-------+
|  1 | a,b,c |
|  2 | x,y,z |
|  3 | 1,3   |
+----+-------+
3 rows in set (0.00 sec)
```

変換

```
+---------+
| cut_col |
+---------+
| a       |
| b       |
| c       |
| x       |
| y       |
| z       |
| 1       |
| 3       |
+---------+
8 rows in set (0.01 sec)

+----+---+---------+
| id | n | cut_col |
+----+---+---------+
|  1 | 1 | a       |
|  1 | 2 | b       |
|  1 | 3 | c       |
|  2 | 1 | x       |
|  2 | 2 | y       |
|  2 | 3 | z       |
|  3 | 1 | 1       |
|  3 | 2 | 3       |
+----+---+---------+
8 rows in set (0.01 sec)
```
